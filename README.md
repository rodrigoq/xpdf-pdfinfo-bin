# xpdf-pdfinfo-bin

**pdfinfo** binaries x64 windows and linux from xpdf.

https://www.xpdfreader.com/

## Installation

```shell
composer require --prefer-dist rodrigoq/xpdf-pdfinfo-bin 
```

## Usage

Binaries of **pdfinfo** lib stored on @vendor/bin

```shell
vendor/bin/pdfinfo
```
